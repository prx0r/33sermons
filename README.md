# 33 Sermons on Nibbāna

**Nibbāna – The Mind Stilled**

A series of 33 sermons by Bhikkhu K. Ñāṇananda, delivered at Meetirigala Nissarana Vanaya Meditation Centre (1988–1991).

Each sermon is presented here with:
- The original teaching (worded intact)
- Plain-English notes on key Pali terms
- Meditations that reveal the logic behind the teaching
